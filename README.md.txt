# Sistema de streaming simples (CineVault)

Projeto desenvolvido como trabalho a fim aprimorar o desenvolvimento e familiarização visual no framework Flutter. 

## Interface

O sistema conta com 8 filmes no catálogo, o usuario pode rolar a página para cima e baixo, ao clicar pode adentrar e ver as informações do filme que deseja.

## Funcionalidades

- Ao clicar no botão "Curtir este Filme", o usuário pode deixar uma deixar uma curtida no filme que deseja.

- Ao clicar em cima, o usuário pode visualizar informações sobre o filme que tenha adentrado, como Sinopse, Direção e Elenco Principal.

- Ao clicar no botão abaixo "Voltar para a lista" o usuário será redirecionado devolta para a lista de filmes(página principal).

## Tecnologias

- Framework Flutter
- Linguagem de programação main.dart
